# Juwina-Travels

This is a fully functional travel agency website built using HTML, CSS, and a little JavaScript.  

It's a travel agency project where user can view travel package and book, they can hire a cab along with driver they can choose or hire for their trip on their occasion travel.

## Features
- Home page with hero and services
- Packages page with “Book Now”
- Cab booking and driver hire
- Contact form
- Success page after booking
